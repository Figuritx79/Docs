# Estructura de datos

Es un forma de almacenar, ordenar, buscar e insertar informacion compleja de forma eficiente en un programa.

Pero antes de empezar creo que es importante dar una deficion a la palabra nodo: 

### Que es un nodo? 

Un nodo es un unidad basica de las estructuras de datos como : listas enlazadas, arboles, grafos, etc.
